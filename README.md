## - Installation <br>
> **Note**
>
> Works only for xterm-256
```console
$ wget -qq "https://raw.githubusercontent.com/RodricBr/Cols/main/cols" -O cols
$ chmod u+x cols
```

Or, just to test Cols:
```console
source <(curl -sk "https://raw.githubusercontent.com/RodricBr/Cols/main/cols")
```
